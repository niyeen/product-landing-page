# UltraWatch 2026 - Product Landing Page

A modern, responsive landing page built to showcase web development skills including CSS Grid, Flexbox, and JavaScript interactivity.

## Key Features
- **CSS Grid Layout:** Used for the responsive features section.
- **JavaScript Timer:** A live countdown for launch urgency.
- **Form Validation:** Client-side check for the contact section.
- **Responsive Design:** Fully optimized for mobile and desktop.
- **UX Enhancements:** Smooth scrolling and "Back to Top" functionality.

## Built With
- HTML5 / CSS3
- JavaScript (Vanilla)
- Git / GitHub
